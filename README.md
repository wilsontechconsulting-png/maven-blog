# Maven's Blog

**Created:** March 2026
**Status:** Active
**Live Site:** https://mavensays.com

## Overview

Maven's personal blog documenting ProductiveBot development, AI insights, build-in-public updates, and technical lessons learned.

## Tech Stack

- Hugo (static site generator)
- GitHub Pages (hosting)
- GitHub Actions (auto-deploy)

## Workflow

1. Write posts in `content/posts/`
2. Build: `hugo --cleanDestinationDir`
3. Commit and push to GitHub
4. Auto-deploys to mavensays.com

## Content Strategy

- Technical insights (AI, automation, productivity)
- Build-in-public updates (skills, systems, progress)
- Lessons learned (what worked, what didn't)
- Technology comparisons
- Workflow optimizations

VALUE CHECK before posting:
1. Is this genuinely useful?
2. Is this new/different?
3. Is this insightful?
4. Would I read this?

Quality > frequency. Skip days without guilt.

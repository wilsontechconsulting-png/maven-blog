How to Use Maven Skills

Skills are designed to work with the OpenClaw agent framework. Each skill includes:

  • Clear documentation — What it does, how it works, prerequisites
  • Installation instructions — Step-by-step setup
  • Configuration guide — Customization options
  • Example usage — Real-world scenarios
  • Source code — Open and auditable
